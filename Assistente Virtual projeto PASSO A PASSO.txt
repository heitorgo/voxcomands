Tutorial:
https://www.youtube.com/watch?v=HJuhrBm8AC8&list=PL39zyvnHdXh8fkuWqzucS93REYaNEJVRx&index=38
O projeto do vídeo usa o reconhecedor vosk
Para instala-lo verifique se o python está na versão 3.8 ou mais recente
> python --version
Instale o vosk
> pip install vosk
Também será necessário o pyaudio e sounddevice
Para baixa-los temos que ir no site:
https://www.lfd.uci.edu/~gohlke/pythonlibs/
Após entrar no site localize os links para o download compatíveis com seu python
No caso desta máquina em que estou no momento cujo python é 3.9 seria o arquivo:
PyAudio-0.2.11-cp39
sounddevice-0.4.4-cp39
Note que existem dois arquivos com esse nome, amd64 e win32, escolha o que for compatível
Agora instale o pyaudio usando o caminho completo do arquivo, no meu caso:  
> pip install C:\Users\Weslei\Downloads\PyAudio-0.2.11-cp39-cp39-win_amd64.whl
De modo genérico podemos exemplificar assim:
> pip install (Caminho do absoluto arquivo)
Vamos testar o vosk API, copie o código da página:
https://github.com/alphacep/vosk-api/blob/master/python/example/test_microphone.py
Cole no arquivo main.py
Baixe o modelo de sua linguagem 
https://alphacephei.com/vosk/models
Descompacte e renomeie a pasta para model e jogue no projeto
Rode o projeto
>python main.py
Baixe o pyttsx3
> pip install pyttsx3
Caso ocorra algum erro baixe o pypiwin32
> pip install pypiwin32
Baixe o tensor flow
> pip install tensorflow
